Sobre este Projeto

Projeto Educacional

Este site foi desenvolvido como parte de um projeto de aprendizado em desenvolvimento web, com foco em HTML, CSS e JavaScript. Não se trata de uma loja real de cartas de Yu-Gi-Oh!, mas sim de um exercício prático para aplicar conceitos de programação web.

Objetivos do Projeto

  • Praticar a estruturação de páginas web com HTML5 semântico
  • Aplicar estilização avançada com CSS3
  • Implementar responsividade para diferentes tamanhos de tela
  • Criar uma experiência de usuário agradável e intuitiva

⚠️ Aviso Importante

Este é um site fictício criado apenas para fins educacionais. As imagens e marcas registradas utilizadas pertencem aos seus respectivos proprietários. Não estamos afiliados, associados ou endossados pela Konami ou pela franquia Yu-Gi-Oh! de forma alguma.

Obrigado por visitar este projeto! Se você está aprendendo desenvolvimento web, espero que este código possa servir de inspiração para seus próprios projetos.
